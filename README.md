# Python Utilities & Script Collection 

A focused collection of lightweight, functional Python utilities designed to handle data structures, algorithmic tracking, and programmatic mathematical mappings.

## Utilities Included

### 1. Caesar Cipher Engine 
* **Features:** A cryptographic tool executing modular arithmetic formulas to securely encrypt (`encode`) or decrypt (`decode`) alphabetical text inputs. Includes automatic non-alphabet character filtering, multi-round session resets, and custom terminal banner assets.
* **Core Concepts:** String manipulation, ASCII character tracking, modular arithmetic mapping, and procedural loop control.

### 2. Functional Dictionary Calculator 
* **Features:** A loop-driven command-line calculator that uses dictionary data structures to map dynamic user operators (`+`, `-`, `*`, `/`) directly to localized math functions. Supports real-time memory persistence to carry over resulting data between calculations.
* **Core Concepts:** First-class functions, associative arrays (dictionaries), stateful loop iterations, and data casting.

### 3. Procedural Password Generator 
* **Features:** An automated security tool compiling secure character sets from variable letter, symbol, and digit counts. Leverages pseudo-random distribution algorithms to thoroughly shuffle arrays, mitigating predictable generation patterns.
* **Core Concepts:** Array concatenation, loop range handling, and the `random` module shuffle algorithms.

### 4. Secret Blind Auction System 🏛️
* **Features:** A local multi-user bidding manager built to handle real-time input collection while clearing historical terminal frames to maintain absolute privacy between distinct bidders. Evaluates max-value algorithms over associative user dictionary containers to calculate the highest bidder upon checkout.
* **Core Concepts:** Dictionary mapping, operating system platform flushes (`os.system`), linear search evaluation, and structure state tracking.
